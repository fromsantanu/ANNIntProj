## Chapter 1: Introduction

In this project, we will build a simple **Artificial Neural Network (ANN)** using PyTorch to solve a real-world problem called **customer churn prediction**.

---

### 🧠 What is Customer Churn?

Customer churn means:
👉 A customer stops using a service.

For example:

* A mobile user stops recharging their SIM
* A subscriber cancels a streaming service
* A bank customer closes their account

For a company, this is important because:
👉 Losing customers = Losing revenue

---

### 🎯 What are We Trying to Do?

We want to build a model that can answer this question:

> “Will this customer leave or stay?”

So the model will take customer details as input and give an output like:

* **0 → Customer will stay**
* **1 → Customer will leave**

---

### 🧩 Why Use ANN?

Think of ANN like a smart decision system.

A human might think like this:

* High bill + many complaints → may leave
* Long-term customer + stable usage → may stay

An ANN learns this kind of pattern automatically from data.

👉 It finds hidden relationships that are not obvious.

---

### 🏗️ What Will We Build in This Project?

Step by step, we will:

1. Understand the dataset
2. Prepare the data (cleaning and converting)
3. Build an ANN model
4. Train the model using data
5. Check how well it predicts
6. Improve the model

---

### ⚙️ Why Use PyTorch?

We will use PyTorch because:

* It is simple to learn
* It gives full control over the model
* It is widely used in real-world AI systems

👉 Think of PyTorch as a toolkit that helps us build and train neural networks easily.

---

### 📌 What You Will Learn

By the end of this project, you will understand:

* How data is used to train a model
* How an ANN is structured (layers and neurons)
* How predictions are made
* How to improve model performance

---

### 🚀 Simple Real-Life Analogy

Imagine you are a shop owner.

After observing customers for months, you start guessing:

* “This person may not come back”
* “This one is loyal”

👉 ANN does the same thing—but using data and mathematics.
