## Chapter 7.2: Activation Functions

Now that we understand the structure of our ANN, the next important concept is:

👉 **Activation Functions**

---

### 🧠 What is an Activation Function?

An activation function decides:

> “Should this neuron pass its signal forward or not?”

---

### 🎯 Why Do We Need It?

Without activation functions:

👉 The ANN behaves like a **simple linear equation**

That means:

* It cannot learn complex patterns
* Adding more layers won’t help much

👉 Activation functions add **non-linearity (complex thinking)**

---

## 🛠️ Types of Activation Functions Used

---

### 🔹 1. ReLU (Rectified Linear Unit)

Used in **hidden layers**

f(x) = \max(0, x)

#### 🧠 Meaning:

* Negative → becomes 0
* Positive → stays same

---

### 🔹 2. Sigmoid Function

Used in the **output layer**

\sigma(x) = \frac{1}{1 + e^{-x}}

#### 🧠 Meaning:

* Converts any value into **0 to 1 (probability)**

---

## ⚙️ Python Program (PyTorch Example)

Now let’s see how we actually use these in code using PyTorch

---

### 🧪 Full Model with Activation Functions

```python id="y5nhyw"
import torch
import torch.nn as nn

class ChurnModel(nn.Module):
    def __init__(self):
        super().__init__()

        self.layer1 = nn.Linear(10, 64)
        self.layer2 = nn.Linear(64, 32)
        self.layer3 = nn.Linear(32, 16)
        self.output = nn.Linear(16, 1)

        # Activation functions
        self.relu = nn.ReLU()
        self.sigmoid = nn.Sigmoid()

    def forward(self, x):
        x = self.relu(self.layer1(x))   # Hidden Layer 1
        x = self.relu(self.layer2(x))   # Hidden Layer 2
        x = self.relu(self.layer3(x))   # Hidden Layer 3
        x = self.sigmoid(self.output(x))  # Output Layer
        return x
```

---

### 🧠 Step-by-Step Understanding

#### 🔹 `nn.ReLU()`

* Applied after each hidden layer
* Removes negative values

---

#### 🔹 `nn.Sigmoid()`

* Applied at the final layer
* Converts output into probability

---

### 📌 Alternative (Short Version Using Sequential)

```python id="c9f0o1"
model = nn.Sequential(
    nn.Linear(10, 64),
    nn.ReLU(),

    nn.Linear(64, 32),
    nn.ReLU(),

    nn.Linear(32, 16),
    nn.ReLU(),

    nn.Linear(16, 1),
    nn.Sigmoid()
)
```

---

### 📈 Real-Life Analogy

* ReLU → like ignoring negative or useless signals
* Sigmoid → like giving a final decision probability

👉 Example:

* “There is 80% chance this customer will leave”

---

### 🚀 Key Takeaways

✔ Activation functions make ANN powerful
✔ ReLU → used inside the network
✔ Sigmoid → used at the final output
✔ Easy to implement in PyTorch
