## Chapter 8: Loss Function and Optimizer

Now that our ANN model is ready, the next question is:

👉 **How does the model learn?**

The answer depends on two key components:

* **Loss Function** → tells how wrong the model is
* **Optimizer** → helps improve the model

---

## 🧠 1. What is a Loss Function?

A loss function measures:

> “How far the model’s prediction is from the actual answer”

---

### 🎯 Simple Example

| Actual | Predicted |
| ------ | --------- |
| 1      | 0.9       |
| 0      | 0.2       |

👉 Small difference → small loss (good)

| Actual | Predicted |
| ------ | --------- |
| 1      | 0.1       |

👉 Large difference → large loss (bad)

---

## 🔹 Binary Cross Entropy (Used in Our Project)

Since our problem is:
👉 **0 or 1 (binary classification)**

We use:

L = -\left[y\log(p) + (1-y)\log(1-p)\right]

---

### 🧠 Meaning (Simple)

* If prediction is close to actual → loss is small
* If prediction is far → loss is large

👉 The model tries to **minimize this loss**

---

## ⚙️ PyTorch Code for Loss Function

```python id="s3j1pq"
import torch.nn as nn

criterion = nn.BCELoss()
```

---

## 🧠 2. What is an Optimizer?

Optimizer decides:

> “How to update the model weights to reduce the loss”

---

### 🎯 Simple Idea

* Model makes prediction
* Loss is calculated
* Optimizer adjusts weights
* Model improves

👉 This repeats many times

---

## 🔹 Adam Optimizer (Used in This Project)

Adam is a popular optimizer because:

✔ Fast
✔ Efficient
✔ Works well in most cases

---

### ⚙️ PyTorch Code for Optimizer

```python id="z4yqbn"
import torch.optim as optim

optimizer = optim.Adam(model.parameters(), lr=0.001)
```

---

### 📌 What is Learning Rate (`lr`)?

Learning rate controls:

> “How big a step the model takes while learning”

---

### 🎯 Example

* Large learning rate → fast but may overshoot
* Small learning rate → slow but stable

👉 Common value:

* `0.001`

---

## 🧩 How Loss and Optimizer Work Together

Step-by-step:

1. Model predicts output
2. Loss function calculates error
3. Optimizer updates weights
4. Model improves

👉 This cycle repeats many times

---

## 📈 Real-Life Analogy

Imagine learning to throw a ball into a basket:

* Loss → how far the ball missed
* Optimizer → how you adjust your throw next time

👉 Over time, you improve accuracy

---

## 🚀 Key Takeaways

✔ Loss function measures error
✔ Binary Cross Entropy is used for 0/1 problems
✔ Optimizer updates the model
✔ Adam is a powerful and commonly used optimizer
