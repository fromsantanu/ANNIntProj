## Chapter 9.1: Training Loop

In the previous chapter, we understood what training means.
Now we focus on the **core mechanism that makes training happen**:

👉 **The Training Loop**

---

### 🧠 What is a Training Loop?

A training loop is:

> “A set of steps repeated many times to help the model learn”

---

### 🎯 Why Do We Need a Loop?

The model does not learn in one step.

👉 It improves gradually by:

* Making mistakes
* Correcting them
* Trying again

---

### 🔁 Basic Flow of Training Loop

Each iteration follows this sequence:

1. Forward pass (prediction)
2. Compute loss
3. Backward pass (find error direction)
4. Update weights

👉 This repeats for many epochs

---

## ⚙️ Training Loop in PyTorch

---

### 🧪 Complete Code Example

```python id="nydf20"
epochs = 20

for epoch in range(epochs):

    # Step 1: Forward Pass
    predictions = model(X_train)

    # Step 2: Compute Loss
    loss = criterion(predictions, y_train)

    # Step 3: Backward Pass
    optimizer.zero_grad()
    loss.backward()

    # Step 4: Update Weights
    optimizer.step()

    # Print progress
    print(f"Epoch {epoch+1}, Loss: {loss.item():.4f}")
```

---

### 🧠 Step-by-Step Explanation

---

#### 🔹 Step 1: Forward Pass

```python id="7r8r3r"
predictions = model(X_train)
```

👉 Input data goes through the model
👉 Output predictions are generated

---

#### 🔹 Step 2: Loss Calculation

```python id="9g2zzr"
loss = criterion(predictions, y_train)
```

👉 Compare prediction with actual values
👉 Calculate how wrong the model is

---

#### 🔹 Step 3: Backward Pass

```python id="i6my3u"
optimizer.zero_grad()
loss.backward()
```

👉 Compute gradients
👉 Understand how to reduce error

---

#### 🔹 Step 4: Update Weights

```python id="z7c7wx"
optimizer.step()
```

👉 Adjust model parameters
👉 Improve predictions

---

### 📌 Important Note

👉 Always call:

```python id="y3p52s"
optimizer.zero_grad()
```

Otherwise:

* Gradients will accumulate
* Training becomes incorrect

---

### 📈 Real-Life Analogy

Think of learning to hit a target:

1. Aim → throw (prediction)
2. Check error (loss)
3. Adjust aim (backpropagation)
4. Try again (update weights)

👉 Repeat until accuracy improves

---

### 🚀 Key Takeaways

✔ Training loop is the heart of learning
✔ It repeats multiple times (epochs)
✔ Each cycle improves the model
✔ Loss should decrease over time
