## Chapter 10.3: F1 Score

In the previous section, we saw:

* Precision → correctness of positive predictions
* Recall → how many actual positives we captured

👉 But sometimes, we need **one single metric** that balances both.

This is where the **F1 Score** comes in.

---

### 🧠 What is F1 Score?

F1 Score means:

> “A balanced measure that combines Precision and Recall”

---

### 📊 Formula

$F1 = 2 \times \frac{\text{Precision} \times \text{Recall}}{\text{Precision} + \text{Recall}}$

---

### 🎯 Why Use F1 Score?

* Precision alone is not enough
* Recall alone is not enough

👉 F1 Score gives a **balance between both**

---

### 🧪 Example

Suppose:

* Precision = 0.8
* Recall = 0.6

👉 F1 Score:


$F1 = 2 \times \frac{0.8 \times 0.6}{0.8 + 0.6} = 0.6857$


👉 So F1 ≈ **0.69**

---

### 📌 Key Insight

F1 Score becomes high **only when both precision and recall are high**

👉 If one is low, F1 drops

---

### ⚠️ When is F1 Important?

Use F1 Score when:

* Data is imbalanced
* Both false positives and false negatives matter

👉 Example:

* Missing churn customer (FN) → loss
* Wrongly targeting loyal customer (FP) → cost

---

### ⚙️ Python Code Example

```python id="c6m1pt"
from sklearn.metrics import f1_score

y_pred = (model(X_test) > 0.5).float()

f1 = f1_score(y_test, y_pred)

print("F1 Score:", f1)
```

---

### 📈 Real-Life Analogy

Think of a student:

* Precision → how accurate answers are
* Recall → how many questions attempted correctly

👉 F1 Score = overall performance balance

---

### 🚀 Key Takeaways

✔ F1 Score combines precision and recall
✔ Useful when data is imbalanced
✔ Gives a balanced evaluation
✔ High F1 means good overall performance
