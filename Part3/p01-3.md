## Chapter 7.3: Dropout Layer

As our model becomes deeper and more powerful, a new problem can appear:

👉 **Overfitting**

To solve this, we use an important technique called:

> **Dropout**

---

### 🧠 What is Overfitting?

Overfitting means:

> “The model memorizes training data instead of learning general patterns”

---

### 🎯 Example

* Model performs very well on training data
* But performs poorly on new (test) data

👉 This is not useful in real life

---

### 🧠 What is Dropout?

Dropout means:

> “Randomly turning off some neurons during training”

---

### 🔍 How It Works

During training:

* Some neurons are **temporarily ignored**
* Next time, different neurons are ignored

👉 So the model cannot depend too much on any single neuron

---

### 📌 Simple Idea

Instead of using all neurons:

👉 Use only a **random subset** each time

---

### 🎯 Why Dropout Helps

✔ Prevents memorization
✔ Forces model to learn better patterns
✔ Improves generalization

---

### ⚙️ Dropout in PyTorch

We use:

```python
nn.Dropout(p)
```

Where:

* `p` = probability of dropping neurons

Example:

* `p = 0.3` → 30% neurons will be turned off

---

### 🧪 Code Example

```python id="r7zqsa"
import torch
import torch.nn as nn

model = nn.Sequential(
    nn.Linear(10, 64),
    nn.ReLU(),
    nn.Dropout(0.3),

    nn.Linear(64, 32),
    nn.ReLU(),
    nn.Dropout(0.3),

    nn.Linear(32, 16),
    nn.ReLU(),

    nn.Linear(16, 1),
    nn.Sigmoid()
)
```

---

### 🧠 Important Note

👉 Dropout works **only during training**

During testing:

* All neurons are used
* No dropout is applied

---

### 📈 Real-Life Analogy

Imagine a classroom:

* Sometimes a few students are absent
* Teacher still teaches

👉 So every student learns independently

This makes learning stronger.

---

### ⚠️ Choosing Dropout Value

Common values:

* 0.2 → light dropout
* 0.3 → moderate
* 0.5 → strong

👉 Too much dropout can reduce learning

---

### 🚀 Key Takeaways

✔ Dropout prevents overfitting
✔ Randomly disables neurons during training
✔ Makes model more robust
