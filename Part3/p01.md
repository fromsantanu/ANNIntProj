## Chapter 7: Building the ANN Model

Now we reach the core part of this project:

👉 **Building the Artificial Neural Network (ANN)** using PyTorch

---

### 🧠 What is an ANN (Simple View)?

An ANN is a system of **connected layers of neurons**.

👉 It works like this:

> Input → Processing → Output

---

### 🧩 Structure of Our Model

We will build a **multi-layer network**:

* Input Layer → receives customer data
* Hidden Layers → learn patterns
* Output Layer → predicts churn

---

### 🏗️ Model Architecture

For our problem, we will use:

* Input Layer → depends on number of features (say 10)
* Hidden Layer 1 → 64 neurons
* Hidden Layer 2 → 32 neurons
* Hidden Layer 3 → 16 neurons
* Output Layer → 1 neuron

---

### 🎯 Why Multiple Layers?

Simple problems need simple models.
But churn prediction is more complex.

👉 More layers help the model:

* Learn deeper patterns
* Capture hidden relationships

---

### ⚙️ Step-by-Step Model Creation

In PyTorch, we define a class for the model.

---

### 🧪 Code Example

```python id="nqq9e5"
import torch
import torch.nn as nn

class ChurnModel(nn.Module):
    def __init__(self):
        super().__init__()

        self.network = nn.Sequential(
            nn.Linear(10, 64),
            nn.ReLU(),

            nn.Linear(64, 32),
            nn.ReLU(),

            nn.Linear(32, 16),
            nn.ReLU(),

            nn.Linear(16, 1),
            nn.Sigmoid()
        )

    def forward(self, x):
        return self.network(x)
```

---

### 🧠 Understanding the Code Simply

#### 🔹 `nn.Linear(a, b)`

* Connects layer with `a` inputs to `b` neurons

👉 Like wires connecting neurons

---

#### 🔹 `ReLU()`

* Removes negative values
* Keeps positive values

👉 Helps model learn better patterns

---

#### 🔹 `Sigmoid()`

* Converts output into probability (0 to 1)

👉 Perfect for yes/no problems like churn

---

### 📌 Flow Inside the Model

1. Input enters the first layer
2. Data gets transformed through hidden layers
3. Final output gives probability

---

👉 ANN works in the same layered way.
