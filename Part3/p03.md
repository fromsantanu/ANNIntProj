## Chapter 9: Training the Model

Now we reach the stage where the model actually **learns from the data**.

👉 This process is called **training**

---

### 🧠 What is Training?

Training means:

> “Showing data to the model again and again so it can learn patterns”

---

### 🎯 What Happens During Training?

For each batch of data:

1. Model makes a prediction
2. Loss is calculated (how wrong it is)
3. Gradients are computed
4. Weights are updated using optimizer

👉 This loop repeats many times

---

### 🔁 What is an Epoch?

An **epoch** means:

> “One full pass through the entire training dataset”

---

### 🎯 Example

* Dataset = 1000 rows
* 1 epoch → model sees all 1000 rows once
* 10 epochs → model sees data 10 times

👉 More epochs = more learning (but too many can overfit)

---

## ⚙️ Training Step-by-Step

We use PyTorch for training.

---

### 🧪 Full Training Loop Example

```python id="c5tk1x"
import torch

epochs = 20

for epoch in range(epochs):
    
    # Forward pass
    outputs = model(X_train)
    
    # Compute loss
    loss = criterion(outputs, y_train)
    
    # Backward pass (compute gradients)
    optimizer.zero_grad()
    loss.backward()
    
    # Update weights
    optimizer.step()
    
    # Print progress
    if (epoch+1) % 5 == 0:
        print(f"Epoch [{epoch+1}/{epochs}], Loss: {loss.item():.4f}")
```

---

### 🧠 Understanding Each Step

#### 🔹 Forward Pass

* Input goes through the model
* Output (prediction) is generated

---

#### 🔹 Loss Calculation

* Compare predicted vs actual
* Measure error

---

#### 🔹 Backward Pass

* Calculate gradients (how to improve)

---

#### 🔹 Optimizer Step

* Update weights to reduce loss

---

## 📌 Important Concepts

---

### 🔹 Gradient

Gradient tells:

> “Which direction to move to reduce error”

---

### 🔹 Backpropagation

This is the process of:

* Sending error backward
* Updating all layers

👉 This is how ANN learns

---

### 🔹 Zeroing Gradients

```python
optimizer.zero_grad()
```

👉 Prevents old gradients from mixing with new ones

---

## 📈 Real-Life Analogy

Think of learning cycling:

* Try → fall → adjust → try again

👉 Each attempt = one training step
👉 Many attempts = learning improves

---

## ⚠️ Common Mistakes

* Too many epochs → overfitting
* Too few epochs → underfitting
* Wrong learning rate → unstable training

---

## 🚀 Output of Training

After training:

✔ Model learns patterns
✔ Loss decreases
✔ Predictions improve
