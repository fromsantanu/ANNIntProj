## Chapter 5: Data Preprocessing

After understanding the dataset, the next step is to **prepare the data properly**.

👉 Raw data is usually messy.
👉 ANN models need **clean and numeric data**.

This step is called **Data Preprocessing**.

---

### 🧠 What is Data Preprocessing?

Data preprocessing means:

> “Cleaning and transforming data so that the model can understand it”

---

### 🎯 Why is This Step Important?

Imagine giving:

* Mixed language
* Missing information
* Wrong values

👉 The model will get confused.

So we make the data:
✔ Clean
✔ Consistent
✔ Numeric

---

## 🔹 5.1 Handling Missing Values

Sometimes data is incomplete.

Example:

* Age = missing
* Payment method = blank

### 🛠️ What can we do?

* Replace with average (for numbers)
* Replace with most common value (for categories)
* Remove rows (if too many values are missing)

👉 Simple example:
If most ages are around 40, fill missing with 40.

---

## 🔹 5.2 Encoding Categorical Variables

ANN cannot understand words like:

* “Monthly”
* “Yearly”
* “UPI”

👉 So we convert them into numbers.

### 🛠️ Common Methods

#### Label Encoding

* Monthly → 0
* Yearly → 1

#### One-Hot Encoding

* Monthly → [1, 0]
* Yearly → [0, 1]

👉 Think of it as translating language into numbers.

---

## 🔹 5.3 Feature Scaling

Different columns may have very different values:

* Age → 20 to 60
* Bill → 500 to 5000

👉 Large values can dominate small ones.

### 🛠️ Solution: Scaling

We bring all values into a similar range.

#### Example:

* 500 → 0.1
* 5000 → 1.0

👉 Now all features are balanced.

---

### ⚙️ Tools We Will Use

* **Pandas** → cleaning data
* **Scikit-learn** → encoding and scaling
* Later we convert everything to tensors for PyTorch

---

### 🧪 Simple Workflow

1. Load dataset
2. Handle missing values
3. Convert text to numbers
4. Scale numerical features
5. Prepare final dataset

---

### 📈 Real-Life Analogy

Think of preparing ingredients before cooking:

* Wash vegetables → cleaning data
* Cut into pieces → formatting
* Measure properly → scaling

👉 Only then cooking (training) works well.

---

### 🚀 Output of This Step

After preprocessing, we will have:

✔ Clean data
✔ Only numeric values
✔ Proper scale

👉 Now the data is ready for model training.
