## Chapter 5.1: Handling Missing Values

In real-world datasets, it is very common to find **missing values**.

👉 Some information is simply not recorded or lost.

---

### 🧠 What are Missing Values?

Missing values mean:

> “Some data entries are empty or unknown”

---

### 🧾 Example

| Age | Monthly Bill | Complaints | Payment Method |
| --- | ------------ | ---------- | -------------- |
| 45  | 2000         | 3          | Card           |
| —   | 1500         | 1          | UPI            |
| 30  | —            | 0          | Cash           |

👉 Here:

* Age is missing in row 2
* Monthly Bill is missing in row 3

---

### 🎯 Why Missing Values are a Problem

ANN models (like the ones we build using PyTorch) **cannot handle empty values**.

If we ignore them:

* Model may crash
* Or give wrong predictions

---

### 🔍 How Do Missing Values Occur?

* User did not fill the form
* Data entry error
* System failure
* Optional fields left blank

👉 So it is normal—but must be handled.

---

## 🛠️ Methods to Handle Missing Values

---

### 🔹 1. Remove Missing Data (Deletion)

If only a few rows have missing values:

👉 We can remove those rows.

✔ Simple
❌ But we lose data

---

### 🔹 2. Fill with Mean (For Numbers)

For numerical columns like Age or Bill:

👉 Replace missing value with average

Example:

* Ages: 30, 40, 50 → Mean = 40
* Missing age → replace with 40

✔ Easy and common method

---

### 🔹 3. Fill with Median

Median = middle value

👉 Useful when data has extreme values

Example:

* 20, 25, 30, 200 → Median = 27.5

✔ Better than mean when outliers exist

---

### 🔹 4. Fill with Mode (For Categories)

For text values:

👉 Replace with most frequent value

Example:

* Payment methods: UPI, Card, UPI → Mode = UPI

✔ Best for categorical data

---

### 🔹 5. Use Constant Value

Sometimes we fill with:

* “Unknown”
* 0

👉 Useful when missing itself has meaning

---

## 📌 Choosing the Right Method

| Data Type          | Method           |
| ------------------ | ---------------- |
| Numerical          | Mean / Median    |
| Categorical        | Mode             |
| Large missing data | Consider removal |

---

### 🧪 Simple Code Example

```python
import pandas as pd

# Fill numerical column with mean
df['Age'] = df['Age'].fillna(df['Age'].mean())

# Fill categorical column with mode
df['Payment'] = df['Payment'].fillna(df['Payment'].mode()[0])
```

---

### 📈 Real-Life Analogy

Imagine a school record:

If a student’s marks are missing:

* You may use average marks
* Or ask: “most students got around this score”

👉 That’s exactly what we do here.
