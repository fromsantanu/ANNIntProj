## Chapter 5.3: Feature Scaling

After encoding, all our data is now numeric.
But there is still one important issue:

👉 Different features have **very different ranges**

---

### 🧠 What is Feature Scaling?

Feature scaling means:

> “Adjusting all values so they are in a similar range”

---

### 🧾 Example of the Problem

| Feature      | Value Range |
| ------------ | ----------- |
| Age          | 20 – 60     |
| Monthly Bill | 500 – 5000  |

👉 Here, “Monthly Bill” values are much larger than “Age”.

---

### 🎯 Why This is a Problem

ANN models (built using PyTorch) give more importance to **larger numbers**.

👉 So the model may:

* Focus too much on “Bill”
* Ignore “Age”

This leads to **wrong learning**

---

### 🔍 Goal of Scaling

Bring all features to a **similar level**

👉 So every feature gets fair importance

---

## 🛠️ Common Scaling Methods

---

### 🔹 1. Min-Max Scaling (Most Common)

This method converts values into a range between **0 and 1**

$x' = \frac{x - x_{min}}{x_{max} - x_{min}}$

#### Example:

| Original | Scaled |
| -------- | ------ |
| 500      | 0.1    |
| 5000     | 1.0    |

👉 All values now lie between 0 and 1

---

### 🔹 2. Standardization (Z-score Scaling)

This method transforms data to have:

* Mean = 0
* Standard deviation = 1

$z = \frac{x - \mu}{\sigma}$

👉 Useful when data has wide variation

---

### 📌 When to Use What?

| Method          | When to Use                  |
| --------------- | ---------------------------- |
| Min-Max         | When values have fixed range |
| Standardization | When data is spread widely   |

---

### ⚙️ Code Example

```python id="p0uxo5"
from sklearn.preprocessing import MinMaxScaler

scaler = MinMaxScaler()
df[['Age', 'Bill']] = scaler.fit_transform(df[['Age', 'Bill']])
```

---

### 🧠 What Happens After Scaling?

✔ All features are balanced
✔ Model learns more effectively
✔ Training becomes faster and stable

---

### 📈 Real-Life Analogy

Imagine a race where:

* One person runs 1 km
* Another runs 100 km

👉 Comparison is unfair.

Scaling is like:
👉 Bringing both to the **same distance scale** before comparing.

---

### 🚀 Final Output

After feature scaling:

✔ All values are in similar range
✔ No feature dominates others
✔ Data is ready for model training
