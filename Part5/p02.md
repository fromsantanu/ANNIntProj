## Chapter 14: Mini Projects

Now it’s time to move from guided learning to **real application**.

👉 In this chapter, you will work on **mini projects** using PyTorch

These projects will help you:

* Apply all concepts together
* Think independently
* Build confidence

---

## 🎯 How to Approach Mini Projects

For each project:

1. Understand the problem
2. Prepare the dataset
3. Build ANN model
4. Train and evaluate
5. Improve performance

👉 Follow the same pipeline you learned

---

## 🧪 Mini Project 1: Student Pass/Fail Prediction

---

### 📌 Problem

Predict whether a student will pass or fail based on:

* Study hours
* Attendance
* Previous marks
* Assignment scores

---

### 🎯 Output

* 0 → Fail
* 1 → Pass

---

### 🧠 Learning Focus

* Simple ANN
* Understanding basic patterns
* Evaluation metrics

---
